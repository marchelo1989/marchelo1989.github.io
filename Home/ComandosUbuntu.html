Sistema Operativo:

Comandos Basicos

  • Para actualiza la lista de paquetes disponibles y sus versiones
    • sudo apt update
  • descargado la lista de software disponible y la versión en la que se encuentra
    • sudo apt upgrade
  • Los 2 comando combinados
    • sudo apt update && sudo apt upgrade
  • Para Instalar las dependencias
    • sudo apt -f install

Idiomas del Sistema

  • Terminar de instalar los idiomas del sistema
  • Buscar "soporte de idiomas"
  • al terminar
  • aplicar a todo el sistema
  • Listo

    • Paquetes .dev

  • Ej sudo dpkg -i /Home/Usuario/nombre_de_archivo.dev
    • sudo dpkg -i

    Paquetes run.tar.gz

  • descomprimir el archivo, buscamos el archivo .run para ejecutarlo buscamos el archivo desde la terminal cd "Ubucacion" -dentro de la carpeta -permisos para el archivo
    • sudo chmod 777 "nombre del archivo"
  • ejecutarlo
    • sudo ./"nombre del archivo"

    Paquetes sh

  • Para Instalar el Archivo
    • sh nombre_de_archivo.sh
  • o
    • bash nombre_de_archivo.sh

    Comandos Para la cosola

    • Para mostrar los archivos de la carpeta en el terminal
      • ls -la
    • Para volver al princio de las carpetas
      • cd
    • Para limpiar la terminal
      • clear
